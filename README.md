# Customer Segmentation using K-Means Clustering

## Project Overview

This project uses Machine Learning and K-Means Clustering to segment customers into different groups based on demographic and purchasing behavior.

The application allows users to enter customer information and predict the customer segment using a trained K-Means model.

---

## Features

- Customer Segmentation using K-Means
- Data Preprocessing and Feature Scaling
- Interactive Streamlit Web App
- Real-time Segment Prediction

---

## Dataset Features

- Age
- Income
- Total Spending
- Number of Web Purchases
- Number of Store Purchases
- Number of Web Visits per Month
- Recency

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib
